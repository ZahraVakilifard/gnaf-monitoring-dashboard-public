# Government Organization GNAF Monitoring Dashboard

 

### Project Overview



This project provides an executive dashboard for monitoring organizational progress in the implementation of GNAF-related initiatives across multiple government organizations.



The dashboard enables stakeholders to evaluate task completion, service implementation, organizational rankings, and compliance metrics through interactive Power BI reports.


### Business Problem



Government organizations are required to complete multiple tasks and service integrations related to the National Geospatial Address Database (GNAF).



Decision makers need a centralized view to:



&#x20;  - Monitor implementation progress



&#x20;  - Compare organizations



&#x20;  - Identify underperforming organizations



&#x20;  - Track service readiness



&#x20;  - Measure compliance indicators




### Data Sources



The dashboard was built using multiple Excel worksheets representing individual organizations.



Data domains include:



&#x20;  - Organizational summary metrics



&#x20;  - Task progress tracking



&#x20;  - Service implementation status



&#x20;  - GNAF compliance scores



&#x20;  - Organization ranking metrics
